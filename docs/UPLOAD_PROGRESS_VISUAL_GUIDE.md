## ðŸ“ Component Sizes

### Inline Mode
- Width: 100% of container
- Height: Auto (expands with content)
- Padding: 12px
- Border Radius: 8px

### Global Mode
- Position: Fixed bottom-right

## ðŸŽ¬ Animation Sequence

### Upload Start (0-100ms)
```
1. Component appears (fade in)
2. Progress bar at 0%
3. Status shows "Uploading..."
```

### During Upload (100ms-completion)
```
1. Progress bar animates smoothly
2. Percentage updates every 100ms
3. Bytes update in real-time
4. Smooth transition (300ms ease)
```

### Upload Complete (final 100ms)
```
1. Progress bar fills to 100%
2. Color changes to green
3. Checkmark appears
4. Status: "Completed"
5. Wait 5 seconds
6. Fade out (300ms)
7. Remove from DOM
```
